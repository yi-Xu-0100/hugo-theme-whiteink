## 特点

### 标头

- 根据设置显示菜单选项。
- 适配移动端，给出菜单按钮。

### 框架布局

- 包含两个部分，左侧为主体内容部分，右侧为个人信息部分。
- 左侧主要展示文章内容或者文件列表结构。
- 右侧包含一个搜索框，个人头像，个人简介，和目录。

### 脚标

- 包含版权信息和备案信息。
